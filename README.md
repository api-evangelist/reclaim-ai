## About

Reclaim.ai integrates with Google Calendar and Outlook to automatically protect focus time,
schedule recurring habits, add buffer time around meetings, and provide team scheduling
analytics. The REST API (v0.1) enables programmatic access to tasks, habits, smart meetings,
calendar sync, events, analytics, and webhooks.

API keys are obtained from https://app.reclaim.ai/settings/developer.
